Clase 45 - Frameworks I

ADONIS:

    Framework orientada al web basado en Node. Cuenta con un template standard que presenta gran adaptabilidad. Inspirado en un framework de PHP: Laravel. Trae incluido:

        - Un CLI
        - Autenticacios
        - Sistema MVC
        - Login social
        - Sistema de plantillas
        - Validaciones
        - Servidor de sockets y cliente
        - Soporte para tests
        - ORM con modelos, migraciones, factorys, y seeds

    Se instala con npm i --global @adonisjs/cli

    En la carpeta donde queremos hacerlos hacemos adonis new my-app

    Entramos a la carpeta y hacemos adonis serve --dev

NEST:

    Basado en node y Express, permite construir un backend en TypeScript o JS con el patrón MVC. Ofrece un poderoso marco de trabajo similar a otros frameworks MVC que existen en otros lenguajes. Cuenta con:

        - Lenguaje tipado
        - Separación por módulos
        - Generación automática de entidades a partir de una base de datos
        - ORM
        - Construcción de endpoints a través de decoradores
        - Inyección de dependencias

    Está influenciado por Angular. Cuenta con su propia CLI para generar y facilitarnos las tareas de creación de todos estos elementos

    CLI:

        Fomenta aplicaciones bien estructuradas

    Se instala con npm install @nestjs/cli

    Creo con nest new my-app

    Corro con nest run start:dev
